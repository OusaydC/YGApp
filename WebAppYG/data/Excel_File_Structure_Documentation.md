# Excel File Structure Documentation
## Yield_Statistics_Complete_Analysis.xlsx

This document provides a detailed description of the Excel file structure generated by `yield_statistics_analysis.py`.

---

## File Overview

**File Name:** `Yield_Statistics_Complete_Analysis.xlsx`  
**Location:** Generated in the same directory where the script is run  
**Purpose:** Comprehensive statistical analysis of wheat yield data and yield gaps by variety, province, scenario, and year

---

## File Structure

The Excel file contains **two main sections**:
1. **Yield Statistics Sheets** - Analysis of actual yield levels
2. **Gap Statistics Sheets** - Analysis of yield gaps (differences between scenarios)

---

## PART 1: YIELD STATISTICS SHEETS

### Sheet 1: `Yield_Overall`
**Description:** Overall statistics across all varieties, scenarios, provinces, and years

**Columns:**
- `Count` - Total number of yield observations
- `Mean` - Average yield (t/ha)
- `Std` - Standard deviation (t/ha)
- `Min` - Minimum yield (t/ha)
- `Q25` - 25th percentile (first quartile) (t/ha)
- `Median` - Median yield (50th percentile) (t/ha)
- `Q75` - 75th percentile (third quartile) (t/ha)
- `Max` - Maximum yield (t/ha)

**Location:** First sheet in the workbook

---

**Description:** Statistics grouped by yield scenario (Potential, Water Limited, Calibrated, Unfertilized, Sowing Date, Observed)

## Column Definitions

### Statistical Columns (appear in all sheets):

1. **Count**
   - **Type:** Integer
   - **Description:** Number of observations used in the calculation
   - **Units:** Count
   - **Example:** 150 means 150 yield measurements were used

2. **Mean**
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Arithmetic average of all values
   - **Units:** t/ha (tons per hectare)
   - **Formula:** Sum of all values / Count
   - **Example:** 3.245 means average yield is 3.245 t/ha

3. **Std** (Standard Deviation)
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Measure of variability/spread of the data
   - **Units:** t/ha
   - **Interpretation:** Higher values = more variability
   - **Example:** 0.856 means yields vary by ±0.856 t/ha on average

4. **Min** (Minimum)
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Lowest value in the dataset
   - **Units:** t/ha
   - **Example:** 0.200 means the lowest yield was 0.2 t/ha

5. **Q25** (25th Percentile / First Quartile)
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Value below which 25% of observations fall
   - **Units:** t/ha
   - **Interpretation:** 25% of yields are below this value
   - **Example:** 2.100 means 25% of yields are below 2.1 t/ha

6. **Median** (50th Percentile)
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Middle value when data is sorted
   - **Units:** t/ha
   - **Interpretation:** 50% of yields are below this value, 50% above
   - **Example:** 3.500 means half of yields are below 3.5 t/ha

7. **Q75** (75th Percentile / Third Quartile)
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Value below which 75% of observations fall
   - **Units:** t/ha
   - **Interpretation:** 75% of yields are below this value
   - **Example:** 4.800 means 75% of yields are below 4.8 t/ha

8. **Max** (Maximum)
   - **Type:** Decimal (rounded to 3 decimal places)
   - **Description:** Highest value in the dataset
   - **Units:** t/ha
   - **Example:** 7.200 means the highest yield was 7.2 t/ha

## Grouping Columns

### Variety
- **Type:** Text
- **Values:** Achtar, Arrehane, Radia, Faiza, Bandera
- **Description:** Wheat variety name

### Scenario
- **Type:** Text
- **Values:**
  - `Potential` - Maximum theoretical yield
  - `Water Limited` - Yield limited by water availability
  - `Calibrated` - Calibrated model yield
  - `Unfertilized` - Yield without fertilizer
  - `Sowing Date` - Yield with different sowing dates
  - `Observed` - Actual observed field yields
- **Description:** Type of yield scenario

### Province
- **Type:** Text
- **Values:** Various Moroccan province names
- **Description:** Geographic location of the plot
- **Note:** Only appears if province data is available in source files

### Year
- **Type:** Integer
- **Values:** 2019, 2020, 2021
- **Description:** Year of the observation
- **Note:** Only appears if year data is available in source files

### Gap_Type
- **Type:** Text
- **Values:**
  - `Total Exploitable Yield Gap` = Potential - Observed
  - `Water Limitation Gap` = Potential - Water Limited
  - `Calibrated vs Water Limited Gap` = Calibrated - Water Limited
  - `Fertilizer Response Gap` = Observed - Unfertilized
  - `Sowing Date Impact Gap` = Calibrated - Sowing Date

## Notes

1. **Sheet Availability:** Not all sheets may be present if:
   - Province data is missing → Province-related sheets won't appear
   - Year data is missing → Year-related sheets won't appear
   - Gap calculations fail → Gap sheets won't appear

2. **Data Precision:** All values are rounded to 3 decimal places

3. **Missing Data:** Rows with missing values in key columns are excluded from calculations

4. **File Size:** The file can be large if all data is available (up to 30 sheets)

5. **Column Order:** Grouping columns always appear first, followed by statistical columns in the order listed above
